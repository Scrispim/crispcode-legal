Perguntas frequentes

Como sincronizar dados entre dispositivos?

O Reformei sincroniza automaticamente via iCloud. Certifique-se de que:

  • Você está conectado com a mesma conta Apple ID nos dois dispositivos
  • O iCloud está ativado em Ajustes → [seu nome] → iCloud
  • Você tem conexão com a internet

Como cancelar minha assinatura Premium?

As assinaturas são gerenciadas pela Apple. Para cancelar:

  • Ajustes → [seu nome] → Assinaturas
  • Selecione Reformei Premium e toque em Cancelar Assinatura

O acesso Premium permanece ativo até o fim do período já pago.

Como restaurar minha compra em um novo dispositivo?

Abra o Reformei, acesse o Paywall e toque em "Restaurar compras anteriores". Sua assinatura será reconhecida automaticamente se estiver ativa na mesma conta Apple ID.

Perdi meus dados. Como recuperar?

Os dados são sincronizados com o iCloud. Se os dados sumirem:

  • Verifique se está logado com a mesma conta Apple ID
  • Aguarde alguns minutos — a sincronização pode ter um pequeno delay
  • Acesse Ajustes → [seu nome] → iCloud → Gerenciar Armazenamento e verifique se o Reformei aparece na lista

Como excluir meus dados permanentemente?

Para remover todos os seus dados do Reformei:

  • Exclua o aplicativo do dispositivo (remove os dados locais)
  • Acesse icloud.com → Configurações → Gerenciar Armazenamento → Reformei → Excluir dados
